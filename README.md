## 功能特性

### 顾客

- 用户的 **注册**，**登录**，**注销**
- 用户购买流程（浏览/查询 -> 添加至购物篮 -> 付款 -> 发送电子邮件确认发货）

### 销售

- 商品目录的 **管理**（包括最基本的添加，删除，修改等操作）
- 后台销售统计报表， 销售状态
- 客户的 浏览/购买 **日志** 记录
